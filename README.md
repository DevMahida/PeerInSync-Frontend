# PeerInSync Frontend

This is the frontend for the PeerInSync project, a platform to connect students and alumni.

This project was bootstrapped with [Vite](https://vitejs.dev/) and uses [React](https://react.dev/).

## Getting Started

1.  **Install dependencies:**
    ```bash
    npm install
    ```
2.  **Run the development server:**
    ```bash
    npm run dev
    ```
    This will start the development server, typically at `http://localhost:5173`.

## Available Scripts

In the project directory, you can run:

```bash
npm run dev
```

Runs the app in the development mode.\
Open [http://localhost:5173](http://localhost:5173) to view it in the browser.

The page will reload if you make edits.\
You will also see any lint errors in the console.

```bash
npm run build
```

Builds the app for production to the `dist` folder.\
It correctly bundles React in production mode and optimizes the build for the best performance.

The build is minified and the filenames include the hashes.

```bash
npm run preview
```

Serves the production build locally. This is a good way to check if the production build is working correctly before deploying.

```bash
npm run lint
```

Lints the project files using ESLint.
